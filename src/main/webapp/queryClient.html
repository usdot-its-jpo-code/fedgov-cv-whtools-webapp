    Welcome to the Connected Vehicles GUI!

    •  About the project

    Welcome to the USDOT Connected Vehicles GUI web application. All functions currently reside under the tools menu.

    Features
    • Start, stop and reset the appliation from the tools menu.
    • Leidos' DigitalEdge integration used for storage and data enrichment.
    • Zoom and pan street map using OpenLayers.
    • Marker color based on vehicle speed.
    • Custom and preset boundary selections.
    • jQuery loading of external GeoJSON files coming soon!
    • Framework based on Bootstrap 3, Leaflet, and typeahead.js. Open source, MIT licensed, and available on GitHub
